## 排除简报

凡是“年度报告摘要”，都不属于本程序的分析范围。

## 财报结构

#### A股财报通常分为以下部分：

+ 公司简介和主要财务指标

+ 管理层讨论与分析

+ 公司治理、环境和社会

+ 重要事项

+ 股份变动及股东情况

+ 债券相关情况

+ 审计报告

+ 财务报表

+ 公司基本情况

+ 关联方及关联交易

+ 重要交易和事项

#### 规则：

+ 以以上关键字正则匹配作为该文档分隔点，其它更细的部分自然划分到各节。

+ 关键字为一个别名列表而非单个字符串，由字符串从长到短匹配，列表可以自由添加

+ 以上关键字通常为标题，标题行的特征是：
  + 一行内就这几个字，有可能带有编号
  + 编号有可能是数字，也可能是汉字数字；
  + 也有可能带括号或点号；
  + 转换过程中字中间可能有空白符号；
  + 标题会出现在目录中，我们以“公司简介和主要财务指标”作为第一个划分点，加个偏移可以简单跳过目录；
  + 应该不会出现在竖排文字中

## 记录状态与审查

+ 每分析完一个财报，都要生成metadata统计和记录状态：
  + 成功，失败，摘要-忽略，等等。
  + 识别到哪些分隔点，每一块多大。
  + 重点关注大小为0的块。
